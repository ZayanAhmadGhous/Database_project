# 🌐 Flask SQL Web App (Local Database Integration)

A web application built using **Flask**, where an HTML-based user interface is connected to a backend **SQL database**. The application runs locally and demonstrates full-stack integration between frontend and backend systems.

---

## 🚀 Project Overview

This project showcases how to:

* Connect an HTML frontend with a backend database
* Perform database operations using SQL
* Handle user input through Flask routes
* Run a complete web application locally

It is designed to demonstrate core concepts of backend development and database connectivity.

---

## 🛠️ Tech Stack

* Python
* Flask (Backend Framework)
* SQL (Database)
* HTML / CSS (Frontend UI)
* DB Connection (e.g., SQLite / MySQL connector)

---

## ✨ Features

* 🌐 Interactive HTML-based UI
* 🔗 Backend integration using Flask
* 🗄️ SQL database connectivity
* ➕ Insert, retrieve, and manage data
* 💻 Runs locally on your system

---

## 📂 How It Works

* User interacts with the HTML interface
* Requests are sent to Flask routes
* Flask processes the request and communicates with the SQL database
* Data is stored/retrieved using SQL queries
* Results are displayed back on the UI
